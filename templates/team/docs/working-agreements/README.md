# Working Agreements

このディレクトリは「人 + エージェント」で進めるための運用合意を置く場所。

## ここに置くもの

- チーム体制と役割分担
- 協業時のコミュニケーション手順
- 実装時の共通ルール
- 完了条件・進捗報告・引き継ぎのプロセス

## ここに置かないもの

- 個別 Issue の計画やタスク分解（`docs/plans/`）
- 開発中の学びや試行錯誤ログ（`docs/notes/`）
- 一時的なメモやセッション断片（`docs/tmp/`）

## ドキュメント構成

- `team-collaboration.md`: 体制、役割、連携手順、運用コマンド
- `engineering-rules.md`: 実装時の共通ルール
- `workflow.md`: 作業フロー、完了条件、進捗報告、引き継ぎ
